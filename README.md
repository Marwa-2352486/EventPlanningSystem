# EventPlanningSystem
This Project "Event Planning System" code is written in Python and will be compatible with Python version 3.5 or above. It can be used by using Visual Studio Code or any Code Editor application.

Features:
1: Create Event
2: Add Guest
3: View Guests
4: Search Guest
5: Add Menu Items
6: View Menu
7: Choose Meal
8: Track Meals
9: View Events
10: Add Venue
11: Rate Venue
12: Exit
